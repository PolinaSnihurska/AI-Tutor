## Production Guidelines

### Video Specifications
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30 fps
- **Format**: MP4 (H.264)
- **Audio**: Clear voiceover with background music at -20dB
- **Length**: 5-12 minutes per video
- **Captions**: Include closed captions for accessibility

### Visual Style
- Clean, professional interface recordings
- Highlight cursor movements
- Use callouts for important UI elements
- Smooth transitions between sections
- Brand colors: Primary blue (#2563EB), accent green (#10B981)

### Voiceover Guidelines
- Clear, friendly, conversational tone
- Moderate pace (not too fast)
- Pause between major sections
- Emphasize key points
- Professional but approachable

### Structure
Each video should follow this structure:
1. **Intro** (15-30 sec): What you'll learn
2. **Main Content** (4-10 min): Step-by-step demonstration
3. **Recap** (30-45 sec): Key takeaways
4. **Call to Action** (15 sec): Next steps or related videos
